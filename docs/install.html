M5Stack StickS3 / ESP32-S3

VoxStick Web Installer

直接从浏览器把 VoxStick 固件刷进 M5Stack StickS3。Use this page to flash VoxStick firmware to your StickS3 without installing ESP-IDF or esptool.

  • Firmware v0.1.4
  • Target ESP32-S3
  • Image offset 0x0
请使用桌面版 Chrome 或 Microsoft Edge 打开本页。 请通过 HTTPS 或 localhost 打开本页，Web Serial 需要安全上下文。

烧录完成后 StickS3 不一定会自动重启。Wait until the page says the install is complete, then unplug USB, double-press the side button to power off, and plug USB back in.

烧录步骤 / Flashing Steps

  1. 1. 打开网页 / Open this page 使用桌面版 Chrome 或 Microsoft Edge。Safari、Firefox、iOS 浏览器不支持 Web Serial。Use desktop Chrome or Edge; Web Serial is not available in Safari, Firefox, or iOS browsers.
  2. 2. 接 USB 数据线 / Connect USB-C 用能传数据的 USB-C 线连接 StickS3。如果串口不出现，优先换一根 数据线或电脑 USB 口。Use a data-capable USB-C cable. If no port appears, try another cable or USB port first.
  3. 3. 进入下载模式 / Enter download mode 按住 StickS3 侧边 reset/PWR 键约 2 秒，看到机身内部绿色 LED 闪烁后松开。Hold the side reset/PWR button for about 2 seconds; release it when the internal green LED blinks.
  4. 4. 选择串口并烧录 / Select the port and flash 点击左侧按钮，选择 StickS3 的串口，然后确认安装。Click Connect and flash, choose the StickS3 serial port, then approve the install.
  5. 5. 等待完成 / Wait for completion 页面显示烧录完成前不要拔线。Do not unplug the cable until the page reports that installation has finished.
  6. 6. 重新开机 / Restart the StickS3 StickS3 带电池，网页烧录后可能不会自动重启。拔掉 USB，双击侧边键关机， 再插回 USB；如只是关机后开机，单击侧边键即可。Because StickS3 is battery-backed, it may not auto-reboot after flashing. Unplug USB, double-press the side button to power off, then plug USB back in; a single press powers it on.
  7. 7. 选择麦克风 / Select the microphone 固件启动后，系统里选择 StickS3-Mic 作为输入设备。 After boot, choose StickS3-Mic as the input device if your app does not select it automatically.

Troubleshooting / 故障恢复

  • 看不到串口时，先确认浏览器是桌面 Chrome/Edge，网页是 HTTPS 打开，并换一根 USB-C 数据线。If no port appears, confirm you are using desktop Chrome/Edge over HTTPS, then try another data cable.
  • 仍然看不到串口时，重新执行官方下载模式动作：长按侧边键约 2 秒，内部绿色 LED 闪烁后松开，再点连接。If the port is still missing, enter download mode again: hold the side button for about 2 seconds, release when the green LED blinks, then reconnect.
  • 如果已经运行 VoxStick 固件，也可以在本仓库里执行 bash tools/trigger-download.sh 让设备重启到 ROM 下载模式。If VoxStick is already running, bash tools/trigger-download.sh can reboot it into ROM download mode.
  • 命令行兜底：esptool.py --chip esp32s3 write_flash 0x0 voxstick-full.bin。Command-line fallback: use esptool to write the merged image at offset 0x0.
