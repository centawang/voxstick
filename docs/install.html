M5Stack StickS3 / ESP32-S3

VoxStick Web Installer

直接从浏览器把 VoxStick 固件刷进 M5Stack StickS3。Use this page to flash VoxStick firmware to your StickS3 without installing ESP-IDF or esptool.

  • Firmware v0.1.4
  • Target ESP32-S3
  • Image offset 0x0
请使用桌面版 Chrome 或 Microsoft Edge 打开本页。 请通过 HTTPS 或 localhost 打开本页，Web Serial 需要安全上下文。

烧录完成后 StickS3 不一定会自动重启。Wait until flashing is complete, then unplug USB, double-click the side PWR button to fully power off, and plug USB back in.

Steps / 步骤

  1. Use desktop Chrome or Edge. Safari and iOS browsers do not support Web Serial.
  2. Plug the M5Stack StickS3 into USB-C.
  3. Click the flash button and select the StickS3 serial port.
  4. After flashing, unplug USB, double-click the side PWR button, then plug USB back in.
  5. When the firmware boots, select StickS3-Mic as the input device if needed.

Troubleshooting / 故障恢复

  • If the port does not appear, try another data-capable USB-C cable or reconnect while holding BtnA.
  • Do not long-press the side PWR button for recovery; on StickS3 that can latch BOOT mode. Use unplug + double-click PWR instead.
  • If you already have VoxStick firmware running, bash tools/trigger-download.sh can reboot it into ROM download mode.
  • Command-line fallback: esptool.py --chip esp32s3 write_flash 0x0 voxstick-full.bin.
