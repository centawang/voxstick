VoxStick Web Installer

Flash VoxStick firmware to your M5Stack StickS3 directly from the browser. No ESP-IDF, Python, or esptool setup required.

  • Firmware v0.1.4
  • ESP32-S3
  • Offset 0x0
Use desktop Chrome or Microsoft Edge. Open this page over HTTPS or localhost.

The StickS3 may not reboot automatically after flashing. Unplug USB, double-press the side button to power off, then plug USB back in.

Flashing Steps

  1. Computer: open this page in desktop Chrome or Microsoft Edge.
  2. Hardware: connect the StickS3 to the computer with a data-capable USB-C cable.
  3. Hardware: hold the StickS3 side reset/PWR button for about 2 seconds. Release it when the internal green LED blinks.
  4. This page: click the “Connect and flash VoxStick” button in the flashing panel. When the browser shows the serial-port picker, choose the StickS3 port and approve the install.
  5. Hardware: when the page reports completion, unplug USB, double-press the StickS3 side button to power it off, then plug USB back in to boot the new firmware.
  6. Computer OS / dictation app: after the firmware boots, select StickS3-Mic in the system sound input settings or your dictation app’s microphone settings.

Troubleshooting

  • No port listed: use desktop Chrome/Edge and try another USB-C data cable.
  • Still missing: enter download mode again by holding the side button for about 2 seconds.
  • If VoxStick firmware is already running, bash tools/trigger-download.sh can reboot it into download mode.
  • Command-line fallback: esptool.py --chip esp32s3 write_flash 0x0 voxstick-full.bin
