VoxStick Web Installer

Flash VoxStick firmware to your M5Stack StickS3 directly from the browser. No ESP-IDF, Python, or esptool setup required.

  • Firmware v0.1.4
  • ESP32-S3
  • Offset 0x0
Use desktop Chrome or Microsoft Edge. Open this page over HTTPS or localhost.

The StickS3 may not reboot automatically after flashing. Unplug USB, double-press the side button to power off, then plug USB back in.

Flashing Steps

  1. Open this page in desktop Chrome or Microsoft Edge.
  2. Connect the StickS3 with a data-capable USB-C cable.
  3. Hold the side reset/PWR button for about 2 seconds. Release it when the internal green LED blinks.
  4. Click the button on the left, select the StickS3 serial port, and approve the install.
  5. When the page reports completion, unplug USB, double-press the side button, then plug USB back in.
  6. Select StickS3-Mic as your input device.

Troubleshooting

  • No port listed: use desktop Chrome/Edge and try another USB-C data cable.
  • Still missing: enter download mode again by holding the side button for about 2 seconds.
  • If VoxStick firmware is already running, bash tools/trigger-download.sh can reboot it into download mode.
  • Command-line fallback: esptool.py --chip esp32s3 write_flash 0x0 voxstick-full.bin
